# Iris Classification using K-Nearest Neighbors (KNN)

## 📌 Problem Statement
Classify iris flowers into species based on features like sepal length, sepal width, petal length, and petal width.

---

## ⚙️ Approach
- Loaded Iris dataset
- Performed train-test split
- Applied K-Nearest Neighbors (KNN) algorithm
- Evaluated model using accuracy and confusion matrix

---

## 📊 Results
- Achieved high accuracy on test data
- KNN performed well for this small and well-structured dataset

---

## 🔍 Key Insight
KNN is a simple yet powerful algorithm that classifies data based on similarity (distance between points).

---

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Pandas, NumPy
